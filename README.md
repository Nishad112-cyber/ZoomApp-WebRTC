
# ZoomApp-WebRTC

**ZoomApp-WebRTC** is a simple video conferencing application built using **React** for the frontend, **Node.js + Express** for the backend, and **WebRTC + Socket.IO** for real-time video and audio communication.

---

## What This Project Does
- Allows users to **join or create video meetings** using a meeting code.
- Supports **real-time video and audio communication** between participants.
- Keeps track of **user meeting history** (backend structure ready).
- Simple **login/signup authentication** using JWT (backend structure ready).

---

## How It Works
1. The **frontend** (React) provides the user interface for login, joining meetings, and displaying video streams.
2. The **backend** (Node.js + Express) handles authentication, meeting management, and stores user history in the database.
3. **WebRTC + Socket.IO** manage the real-time communication of video/audio streams between participants.

---

## Tech Stack
- **Frontend:** React, Material UI, React Router
- **Backend:** Node.js, Express, MongoDB, Socket.IO
- **Communication:** WebRTC for video/audio streaming
- **Authentication:** JWT
